# Tauri Examples

Desktop applications built with [Tauri v2](https://v2.tauri.app/) and
[sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx).

## Examples

| Example | Description |
|---------|-------------|
| [non-streaming-speech-recognition-from-file](./non-streaming-speech-recognition-from-file/) | Transcribe audio/video files using offline ASR with VAD. Supports 62 models, audio playback, SRT export, and segment WAV export. |
| [non-streaming-speech-recognition-from-microphone](./non-streaming-speech-recognition-from-microphone/) | Live microphone transcription using offline ASR with VAD. Supports 62 models, wall-clock timestamps, recording playback, SRT export, and segment WAV export. |
